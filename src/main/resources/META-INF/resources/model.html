Tagline in English

string

text

100

M

Description:

Short catchphrase describing the service. (100 characters). Usually displayed close to the service name and should refer to the main value or purpose of the service. NB: Can also be used as the first sentence in the service description. In English.

100

u_description_en

Description in English

String

text

1000

M

Description:

A high-level description in fairly non-technical terms of

a) what the service does, functionality it provides and services or resources it enables to access,

b) the benefit to a user/customer. Which problems are you solving and for whom.

c) recommended use if not mentioned above.

In English.
