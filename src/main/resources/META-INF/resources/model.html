u_description_en

Description in English

String

text

1000

M

Description:

A high-level description in fairly non-technical terms of

a) what the service does, functionality it provides and services or resources it enables to access,

b) the benefit to a user/customer. Which problems are you solving and for whom.

c) recommended use if not mentioned above.

In English.
