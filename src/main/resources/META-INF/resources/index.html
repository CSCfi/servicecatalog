Service catalog API

CSC Services Public API

The CSC service catalog complies with the EOSC standard and the implementation of the FAIR principles (findable, accessible, interoperable, and reusable).

Features of content:

  • Service persistent identifier PID and short name of service: Unique identifiers for quick access.
  • Brief description of the functionality of the service
  • Bilingual descriptions: Available in both Finnish and English, serving a wider audience.
  • Access and Instructions for Use: Clear instructions on requirements, access to and use of services.
  • Support & Contact Information: Direct links to support resources, user guide, and documentation.
